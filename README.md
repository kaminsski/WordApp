# Word-App
 Word-App website is built with the MERN stack. Designed to increase vocabulary knowledge

## Link
https://word-app-kztv.vercel.app/

## Functions/features of the website
* Login/Register
* Add/delete word
* Profile


## Technologies used


* HTML5
* CSS3
* JavaScript (ES6)
* GitHub
* React
* Bootstrap
* MongoDB
* NodeJS
* ExpressJS
* Bcrypt
* JWT
* Validator
